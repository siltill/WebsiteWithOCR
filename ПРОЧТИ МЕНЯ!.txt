ТРЕБОВАНИЯ ДЛЯ ЗАПУСКА ПРОЕКТА

Установка Go
Скачать с официального сайта golang.org/dl
Установить и убедиться, что Go добавлен в PATH
Проверить командой:
go version

Установка Node.js и npm (для Vue.js)
Скачать с сайта nodejs.org
Убедиться, что npm установлен
Проверить командой:
node -v
npm -v

Установка Vue.js (локально в проекте)
Переход в папку frontend и установка зависимостей:
cd frontend/ocr_app_frontend/
npm install
(Vue будет установлен автоматически через package.json)

Установка Poppler
Скачать сборку Poppler для Windows (Приложен к проекту)
Распаковать и добавить путь к папке bin в переменную среды PATH
Пример пути: C:\poppler\Library\bin

Установка Tesseract OCR
Установить Tesseract OCR для Windows (Приложен к проекту)
Добавить путь установки в PATH, если установщик не сделал это автоматически
Пример пути: C:\Program Files\Tesseract-OCR
Проверить командой:
tesseract --version


ЗАПУСК ПРИЛОЖЕНИЯ

Для удобного запуска используется файл start_app.bat.

start_app.bat выполняет:

Запуск бэкенда командой:
go run main.go

Запуск фронтенда командой:
npm run dev
